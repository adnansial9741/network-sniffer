# Network Packet Sniffer

A professional, modular network packet sniffer built in Python. This tool captures and analyzes network traffic in real-time, providing detailed insights into network protocols and data flow.

## Features

- Real-time packet capture with color-coded output
- Protocol analysis: Ethernet, IP, TCP, UDP, ICMP, ARP, HTTP, DNS
- Statistics gathering: protocol distribution, top talkers, port usage
- BPF filter support for targeted capture
- Clean, modular architecture
- Comprehensive packet analysis
- Professional output formatting
